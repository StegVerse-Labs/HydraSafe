# Inspection Checklist (v0.3)

## Monthly
- [ ] Sensor status check
- [ ] E-stop status check
- [ ] Cylinder restraints and signage check
- [ ] Vent/exhaust status evidence updated (if applicable)

## Quarterly
- [ ] Interlock trip test evidence captured
- [ ] Incident log reviewed and trend-noted

## Annual
- [ ] Re-validation / re-commission evidence captured
- [ ] Documentation audit complete
