# Inspection Checklists (v0.1)

## Monthly
- verify sensor functional checks completed
- verify E-stop test record
- verify cylinder restraints & signage
- verify exhaust status (operator evidence)

## Quarterly
- interlock trip test with evidence logging
- review incident log for trends
- review training record currency

## Annual
- full commissioning re-validation (as required by policy/AHJ)
- audit documentation set and update references
