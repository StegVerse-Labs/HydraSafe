# HydraSafe

HydraSafe is the DiamondOps safety + permitting layer for hydrogen/reactive-gas environments.

This repo is product-facing and includes:
- permit packet templates (framework-level)
- commissioning & inspection checklists
- incident response playbooks
- event/export conventions aligned to DiamondOps-Core

## Scope (v0.1)
- Documentation-first “Pod” deliverables
- Compliance templates and runbooks
- Integration specs for YieldOS ingestion

## Dependency
Aligned to DiamondOps-Core schemas and standards.
